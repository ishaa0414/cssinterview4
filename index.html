flex-wrap:this property takes into consideration the width of the flex-containers.If the property is set to wrap ,it will consider the width and when it is set to no wrap, the flex containers shrink to adjust the width of the screen.
There is a shorthand for flex-direction: and flex-wrap.It is known as flex-flow.
